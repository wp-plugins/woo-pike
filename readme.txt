== Description ==

This is WooCommerce plugin. When order origin ip is from Tor network or another network that provide anonymity and is monitored by pike.hqpeak.com then order note is added and 
order is marked with appropriate label. 


== Installation ==

1. Extract `woo-pike`  archive to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check the order list
